# **App Name**: ChessConnect

## Core Features:

- Local Play: Offline Player vs. Player mode on the same device.
- AI Opponent: Offline Player vs. AI with Easy and Hard difficulty levels. The AI will leverage lightweight evaluation techniques for move decisions.
- Realtime Multiplayer: Online multiplayer mode using Firebase Realtime Database for syncing moves between players in real time.
- Room System: Create and join game rooms using unique room codes, facilitated by Firebase Realtime Database.
- Move Validation: Server-side validation of moves to prevent cheating in online multiplayer mode using a tool.
- AdSense Integration: Layout optimized for Google AdSense with a banner ad below the chessboard. Ads are hidden in offline modes.
- Mobile-Friendly UI: Responsive design with a fixed-size chessboard optimized for touch input on mobile devices.

## Style Guidelines:

- Primary color: Dark forest green (#228B22), reflecting the strategy and focus of chess.
- Background color: Soft beige (#F5F5DC) to provide a neutral and calming backdrop.
- Accent color: Gold (#FFD700) for interactive elements and highlights, symbolizing prestige and clarity.
- Body and headline font: 'Inter', a sans-serif for clean and modern readability, and for code snippets.
- Fixed-size chessboard layout that prevents resizing or jitter during gameplay, maintaining a consistent user experience.
- Use of simple, minimalist icons for settings and interactive elements to ensure clarity and ease of use.
- Subtle transitions for piece movements and game state changes to enhance the user experience without being distracting.
